# Bug Prediction using Machine Learning and LLMs

This project explores how machine learning and large language models can be used to analyze software repository data and predict bug-prone commits.

## Features
- Commit classification (bug vs non-bug)
- ML baseline model
- LLM-based classification

## Motivation
This project is part of my preparation for PhD research in Machine Learning for Software Engineering.

## Tech Stack
- Python
- scikit-learn
- Hugging Face Transformers

## Research Questions
- Can machine learning predict bug-prone commits?
- Do LLM embeddings improve performance?

## Methodology
- GitHub data collection
- Text preprocessing
- ML and LLM-based models
